# Okto - Smart Finansnyheder

En personlig finansnyheds-app med AI-indsigter, bygget som interaktiv prototype.

## Filer

- `index.html` — Komplet app-prototype (åbn i browser)
- `okto-mascot.png` — Okto-maskotten (pixel-art blæksprutte med pressehat og briller)

## Kørsel

Åbn `index.html` i en browser. Begge filer skal ligge i samme mappe.

Brug navigationsbjælken i bunden til at skifte mellem skærmbilleder.

## Skærmbilleder

1. **Velkommen** — Splash med maskot og tagline
2. **Tilmeld** — Opret konto
3. **Profil 1–4** — Onboarding: alder, bolig, lån, opsparing
4. **Loading** — Animeret sekvens hvor Okto henter nyhedskilder
5. **Feed** — Personligt nyhedsfeed med AI-indsigter
